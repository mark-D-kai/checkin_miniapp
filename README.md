# 打卡助手 - 微信小程序

一款简洁实用的打卡小程序，帮助您轻松管理日常任务和工作日打卡安排。

## 功能特性

### 📋 任务管理

- **任务创建**: 支持创建每日、仅工作日两种重复频率
- **任务编辑**: 随时修改任务名称、频率和时间范围
- **任务删除**: 不再需要的任务可随时删除
- **任务状态**: 标记任务为已完成或未完成
- **颜色标记**: 支持8种预设颜色分类任务

### 📅 日历视图

- **月历展示**: 直观的月历布局，清晰查看整月任务安排
- **打卡记录**: 标记已完成和未完成的任务
- **工作日识别**: 自动识别工作日（周一至周五）和周末
- **任务筛选**: 按任务类型和状态筛选展示

### 📊 统计分析

- **完成情况统计**: 任务完成率、连续打卡天数、总打卡次数统计
- **月度日历视图**: 用点状标记直观展示每日打卡状态
- **历史记录**: 查询任意月份的打卡记录
- **任务切换**: 支持切换不同任务查看统计数据
- **日期详情**: 点击日期查看当天打卡详情和备注

### ⚙️ 个性化设置

- **数据备份**: 一键导出所有数据为JSON文件
- **数据恢复**: 从备份文件导入恢复数据
- **任务管理**: 集中管理所有任务的启用/禁用状态

## 技术栈

- **框架**: 微信小程序原生开发
- **数据存储**: 小程序本地存储（wx.setStorageSync）
- **图表**: 自定义Canvas图表组件（预留，未启用）
- **日历**: 自定义日历组件
- **工具类**: 封装日期处理、数据操作等通用方法

## 快速开始

### 环境要求

- 微信开发者工具（最新版本）
- 微信小程序开发账号（可选，测试可使用测试号）

### 安装步骤

1. **下载项目**

将项目文件下载到本地目录

2. **导入项目**

- 打开微信开发者工具
- 选择"导入项目"
- 选择项目根目录
- 填入 AppID（如需发布）或选择"测试号"

3. **运行项目**

- 点击"编译"按钮
- 在模拟器中预览效果
- 如需真机测试，点击"预览"扫码打开

## 使用指南

### 今日打卡

1. **首页展示**：打开小程序首页，自动显示今日日期和应打卡任务
2. **一键打卡**：点击任务右侧的"打卡"按钮，快速标记任务完成
3. **标记未完成**：点击"未完成"按钮，记录未完成任务
4. **添加备注**：点击"备注"按钮，添加任务完成情况说明

### 添加任务

1. 点击首页"添加任务"按钮
2. 输入任务名称
3. 选择任务频率（每日/仅工作日）
4. 选择任务颜色（8种预设颜色可选）
5. 设置任务开始和结束时间
6. 点击"保存"完成任务创建

### 日历视图

1. 点击底部导航"日历"进入日历页面
2. 查看整月任务安排和打卡记录
3. 点击日期查看当天任务详情
4. 滑动切换月份查看历史记录

### 统计分析

1. 点击底部导航"统计"进入统计页面
2. 选择要查看的任务
3. 查看连续打卡天数、完成率、总打卡次数
4. 滑动切换月份查看月度打卡日历
5. 点击日期查看当天打卡详情和备注

### 任务管理

1. 点击底部导航"我的"进入个人中心
2. 选择"任务管理"查看所有任务
3. 编辑或删除不再需要的任务
4. 按颜色分类筛选任务

## 数据管理

### 导出数据

1. 进入"设置"页面
2. 选择"导出数据"
3. 数据将导出为JSON文件，可分享给好友或保存到本地

### 导入数据

1. 进入"设置"页面
2. 选择"导入数据"
3. 选择之前导出的JSON备份文件
4. 确认导入，恢复所有任务和打卡记录

## 数据结构

### 任务数据结构

```javascript
{
  id: '唯一标识符',
  name: '任务名称',
  frequency: 'daily/workday',  // 每日/仅工作日
  color: '#4A90D9', // 任务颜色
  startTime: '2024-01-01', // 任务开始日期
  endTime: '2024-01-31', // 任务结束日期
  isActive: true, // 是否启用
  createdAt: '2024-01-01', // 创建时间
  updatedAt: '2024-01-01' // 更新时间
}
```

### 打卡记录结构

```javascript
{
  id: '唯一标识符',
  taskId: '任务ID',
  date: '2024-01-01', // 打卡日期
  status: 'completed/missed', // 打卡状态
  note: '备注信息',
  checkedInAt: '2024-01-01T09:00:00.000Z' // 打卡时间
}
```
