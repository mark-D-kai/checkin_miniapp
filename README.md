# 打卡助手 - 微信小程序

一款简洁实用的打卡小程序，帮助您轻松管理日常任务和假期安排。

## 功能特性

### 📋 任务管理

- 添加、编辑和删除打卡任务
- 设置任务重复周期（每日/每周/每月）
- 标记任务完成状态

### 📅 日历视图

- 直观的月历展示
- 标记已完成和未完成的任务
- 显示节假日信息

### 📊 统计分析

- 任务完成情况统计
- 可视化图表展示
- 历史打卡记录查询

### 🎯 假期管理

- 导入自定义假期数据
- 自动识别法定节假日
- 灵活的假期设置

### ⚙️ 个性化设置

- 自定义主题颜色
- 提醒时间设置
- 数据备份与恢复

## 技术栈

- **框架**: 微信小程序原生开发
- **数据存储**: 小程序本地存储
- **图表**: 自定义图表组件
- **日历**: 自定义日历组件

## 快速开始

### 环境要求

- 微信开发者工具
- 微信小程序开发账号

### 安装步骤

1. **克隆项目**

```bash
git clone https://gitee.com/yourusername/checkin_miniapp.git
```

1. **导入项目**

- 打开微信开发者工具
- 选择"导入项目"
- 选择项目根目录
- 填入 AppID（如需发布）或选择"测试号"

1. **运行项目**

- 点击"编译"按钮
- 在模拟器中预览效果

## 使用指南

### 添加任务

1. 点击首页的"添加任务"按钮
2. 填写任务名称和描述
3. 设置任务重复周期
4. 保存任务

### 打卡记录

1. 在日历页面点击对应日期
2. 选择要打卡的任务
3. 标记为已完成

### 统计查看

1. 点击底部导航的"统计"页面
2. 查看任务完成情况
3. 切换不同的统计维度

## 自定义配置

### 导入节假日

1. 进入"设置"页面
2. 选择"导入节假日"
3. 上传JSON格式的假期数据

### 主题设置

1. 进入"设置"页面
2. 选择"主题颜色"
3. 自定义导航栏和背景色

## 数据结构

### 任务数据结构

```javascript
{
  id: '唯一标识符',
  name: '任务名称',
  description: '任务描述',
  repeat: 'daily/weekly/monthly',
  days: [1, 2, 3], // 重复星期几（仅weekly）
  date: '2024-01-01', // 具体日期（仅monthly）
  created: '创建时间',
  updated: '更新时间'
}
```

### 打卡记录结构

```javascript
{
  taskId: '任务ID',
  date: '打卡日期',
  completed: true/false,
  time: '打卡时间'
}
```
